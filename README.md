# My OpenClaw Personal Project

一个基于OpenClaw的个人AI助手项目，用于学习和实验。

## 🚀 项目目标
- 学习OpenClaw架构和开发
- 创建自定义技能和插件
- 集成到个人工作流中
- 探索AI助手在不同场景的应用

## 📁 项目结构
```
my-openclaw/
├── README.md          # 项目说明
├── package.json       # Node.js项目配置
├── src/              # 源代码目录
├── skills/           # 自定义技能
├── plugins/          # 自定义插件
└── docs/             # 文档
```

## 🛠️ 技术栈
- **OpenClaw** - AI助手平台
- **Node.js** - 运行时环境
- **JavaScript/TypeScript** - 开发语言
- **GitHub** - 版本控制和协作

## 📦 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 开发模式
```bash
npm run dev
```

### 3. 构建项目
```bash
npm run build
```

## 🔧 开发指南

### 创建自定义技能
1. 在 `skills/` 目录下创建新技能
2. 编写 `SKILL.md` 文件描述技能功能
3. 实现技能逻辑

### 创建自定义插件
1. 在 `plugins/` 目录下创建新插件
2. 遵循OpenClaw插件规范
3. 注册到OpenClaw配置中
