# Hello World Skill

一个简单的示例技能，用于演示如何在OpenClaw中创建自定义技能。

## 功能描述
- 响应"hello"或"你好"消息
- 返回友好的问候语
- 显示当前时间
- 演示技能的基本结构

## 使用方法
当用户发送以下消息时，技能会被触发：
- "hello"
- "你好"
- "hi"
- "打招呼"

## 响应示例
```
用户: hello
助手: 你好！我是你的OpenClaw助手。当前时间是：2026-03-17 09:45
```

## 技能文件结构
```
hello-world/
├── SKILL.md          # 技能说明文档
├── index.js          # 技能主逻辑
└── package.json      # 技能配置（可选）
```

## 开发说明
1. 技能目录必须包含 `SKILL.md` 文件
2. 技能逻辑可以写在 `index.js` 或其他文件中
3. 技能需要注册到OpenClaw配置中才能使用
